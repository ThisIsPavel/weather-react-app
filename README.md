## React Weather

Приложения для отслеживания текущей погоды и погоды на 3 дня, созданное с помощью React

## Функциональность

- Отображение погода - текущая/на 3 дня
- Поиск городов
- Смена темы приложения

## В разработке

- Popup с погодой по часам
